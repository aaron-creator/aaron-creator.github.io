Fishermenadmin Webapp

Fishermenadmin Webapp is a Web Application developed for a Fisheries/Marine department of the Central/State Government of India. The main purpose of this Web application is that it is used by an Admin in any Organization to which it is handed over. The Admin has the control and authority to add the fisherman and to approve him as a valid fisherman who goes into the sea to have his occupation.
